🚀 인터넷에 올리는 방법

GitHub · Vercel · Supabase — 초보자용 안내서

😊 좋은 소식부터!

이번에 힘들었던 계정 만들기 · 로그인 · 서로 연결하기는 대부분 한 번만 하는 일이었어요. 이제 다 되어 있으니, 앞으로는 훨씬 간단합니다.

앞으로 새 앱을 만들면, 아래 초록색(매번) 부분만 하면 됩니다. 주황색(한 번만) 부분은 이미 끝났어요.

⚡ 가장 간단한 요약

Claude에게 이렇게 말하세요: “이거 인터넷에 올려줘.”
1
Claude가 코드를 GitHub(코드 창고)에 자동으로 올립니다. Claude가 함
2
Claude가 Vercel 화면에서 어디를 누르라고 한 걸음씩 알려줍니다. 보통 Import → Deploy 두 번 클릭. 내가 클릭
3
끝! 진짜 인터넷 주소가 생깁니다. (예: ○○○.vercel.app) Claude가 확인
💡 데이터를 저장하는 앱이면 Supabase 단계가 하나 더 있어요. (아래 참고) 저장이 필요 없는 앱이면 이 요약만으로 끝납니다.

✅ 도구(GitHub CLI) 설치 — 이미 완료

🌐 Vercel 매번 조금

이게 뭐예요? GitHub의 코드를 진짜 접속 주소로 만들어 주는 곳이에요.

1
Claude가 알려주면 vercel.com/new 로 들어갑니다. 내가 클릭
2
Import Git Repository 목록에서 방금 만든 앱 이름 옆의 Import 클릭. 내가 클릭
3
설정은 건드리지 말고 아래 검은 Deploy 버튼 클릭. 내가 클릭
4
Congratulations! 화면이 뜨면 성공. 가운데 미리보기 그림을 누르면 진짜 앱이 열립니다. Claude가 확인
💡 “2단계 보안(2FA) 설정할래?” 가 나오면 지금은 Skip securing my account(건너뛰기)를 눌러도 됩니다. GitHub↔Vercel 연결은 이미 해뒀으니 다음부터는 위 4단계만 하면 돼요.

🗄️ Supabase 저장이 필요할 때만

이게 뭐예요? 입력한 내용을 인터넷에 저장해서, 핸드폰·아이패드·PC 어디서나 똑같이 보이게 해주는 곳이에요. (메모·일정·기록처럼 저장이 필요한 앱에만 씁니다.)

✅ 계정·프로젝트 — 이미 있음 (oevdcrdcnleuoqziryan.supabase.co)
1
Claude가 저장할 표(테이블) 만드는 글(SQL)을 줍니다. Supabase의 SQL Editor에 붙여넣고 Run 클릭. 내가 클릭
2
초록색 Connect 버튼을 눌러 나오는 주소와 열쇠값(publishable key)을 복사해 Claude에게 붙여넣기. 내가 복사
3
Claude가 그 값을 앱에 넣고 연결·확인합니다. Claude가 함
💡 열쇠값 중 publishable (또는 anon) 만 씁니다. 이건 공개해도 안전한 값이에요. service_role / secret 이라고 적힌 건 절대 복사하지 마세요. (헷갈리면 화면을 캡처해서 물어보시면 Claude가 골라드려요.)

🛟 막히거나 헷갈릴 때

  • 어떤 화면이든 캡처해서 Claude에게 보여주기 — 그러면 다음에 뭘 눌러야 할지 알려줍니다.
  • 캡처 방법: Windows + Shift + S → 마우스로 영역 드래그 → 대화창에 Ctrl + V
  • 창이 여러 개라 헷갈리면: Alt + Tab 으로 창 바꾸기. 맨 위 탭 이름을 보고 고르면 됩니다.
  • “이거 맞아?” 싶으면 누르기 전에 먼저 캡처해서 물어보세요. 실수해도 대부분 되돌릴 수 있어요.
