🚀 내가 만든 웹앱, 인터넷에 올리는 방법

GitHub · Vercel · Supabase — Claude와 함께라면 개발을 몰라도 OK
주황 = 처음 한 번만 초록 = 앱 만들 때마다 파랑딱지 = Claude가 함 빨강딱지 = 내가 클릭

😊 딱 한 가지만 기억하세요

다 만든 뒤 Claude에게 이렇게 말하세요 —
“이거 인터넷에 올려줘.”

그러면 Claude가 지금 컴퓨터에 무엇이 준비돼 있는지(도구 설치·로그인·연결 상태)를 스스로 확인하고, 빠진 것만 콕 집어 한 걸음씩 알려줍니다. 그러니 “내가 전에 어디까지 했더라?”를 기억 못 해도 괜찮아요. 😊

⚡ 앱 만들 때마다 — 이게 전부예요 매번

1
Claude가 코드를 GitHub(코드 창고)에 자동으로 올립니다. Claude가 함
2
Vercel에서 Import → Deploy 두 번 클릭. (Claude가 어디를 누를지 알려줘요.) 내가 클릭
3
끝! 진짜 인터넷 주소가 생깁니다. (예: ○○○.vercel.app) Claude가 확인
💡 데이터를 저장하는 앱(메모·일정·기록 등)이면 Supabase 단계가 하나 더 있어요. 아래 “Supabase” 부분을 펼쳐 보세요. 저장이 필요 없으면 이 3단계로 끝입니다.

🌐 Vercel에서 내가 누르는 부분 매번

Vercel은 GitHub의 코드를 진짜 접속 주소로 만들어 주는 곳이에요.

1
Claude가 알려주면 vercel.com/new로 들어갑니다. 내가 클릭
2
Import Git Repository 목록에서 방금 만든 앱 옆의 Import 클릭. 내가 클릭
3
설정은 건드리지 말고 검은 Deploy 버튼 클릭. 내가 클릭
4
Congratulations!가 뜨면 성공. 미리보기 그림을 누르면 진짜 앱이 열립니다. 그 주소창의 주소가 공유용 링크예요. Claude가 확인
💡 “2단계 보안(2FA) 설정할래?”가 나오면 Skip securing my account(건너뛰기)를 눌러도 됩니다. (나중에 켜는 걸 권장해요.)
🗄️ Supabase — 저장이 필요한 앱만 필요할 때만

이게 뭐예요? 입력한 내용을 인터넷에 저장해서, 핸드폰·태블릿·PC 어디서나 똑같이 보이게 해줘요. 저장이 필요 없는 앱이면 이 부분은 건너뛰세요.

1
Supabase에서 프로젝트 하나 선택(또는 새로 만들기). 이미 있으면 재사용하면 됩니다. 내가 클릭
2
Claude가 주는 표(테이블) 만드는 글(SQL)을 SQL Editor에 붙여넣고 Run 클릭. 내가 클릭
3
초록 Connect 버튼을 눌러 나오는 주소와 열쇠값(publishable key)을 복사해 Claude에게 붙여넣기. 내가 복사
4
Claude가 앱에 넣고 연결·확인합니다. Claude가 함
⚠️ 열쇠값 중 publishable(또는 anon)만 씁니다 — 공개해도 안전한 값이에요. service_role / secret이라 적힌 건 절대 복사하거나 남에게 주지 마세요. (헷갈리면 화면을 캡처해 Claude에게 물어보면 골라줍니다.)
🧰 처음 한 번만 — 아직 준비가 안 됐다면 최초 1회

이미 계정이 있고 전에 배포해 본 분은 이 부분을 건너뛰어도 됩니다. (Claude가 알아서 확인해요.) 처음이거나 새 컴퓨터라면 아래를 한 번만 해두면 됩니다.

1
계정 3개 만들기 — github.com · vercel.com · supabase.com 에서 무료 가입. (Vercel·Supabase는 “Continue with GitHub”로 가입하면 간단) 내가 가입
2
Claude에게 “필요한 도구를 설치해줘” — Claude가 git과 GitHub 도구를 알아서 설치·확인합니다. Claude가 함
3
GitHub 로그인 — Claude가 안내하면 화면의 코드를 입력하거나 Authorize 클릭. 내가 클릭
4
GitHub ↔ Vercel 연결 — Vercel에서 Continue with GitHub → Authorize. 한 번 연결하면 계속 유지돼요. 내가 클릭
💡 이 부분이 됐는지 안 됐는지 몰라도 괜찮아요. 그냥 “올려줘”라고 하면 Claude가 확인하고, 필요한 것만 이 순서대로 짚어줍니다.

🛟 막히거나 헷갈릴 때

  • 어떤 화면이든 캡처해서 Claude에게 보여주기 — 다음에 뭘 누를지 알려줍니다.
  • 캡처(Windows): Windows + Shift + S → 영역 드래그 → 대화창에 Ctrl + V
  • 창이 여러 개면: Alt + Tab으로 창 바꾸기. 맨 위 탭 이름을 보고 고르세요.
  • “이거 맞아?” 싶으면 누르기 전에 먼저 캡처해서 물어보세요. 실수해도 대부분 되돌릴 수 있어요.

✏️ 이미 올린 앱을 고치고 싶을 때

Claude에게: “글자 크게 해줘 / 색 바꿔줘 / 항목 늘려줘” 등

Claude가 코드를 고쳐 GitHub에 다시 올리면, Vercel이 자동으로 인터넷 주소에 반영합니다. 대부분 아무것도 안 눌러도 됩니다. 😊

개발을 몰라도 괜찮아요 · 막히면 화면을 캡처해서 Claude에게 물어보세요
