🚀 내가 만든 웹앱, 인터넷에 올리는 방법

GitHub · Vercel · Supabase — 개발을 몰라도 되는 초보자용 안내서
주황 = 처음 한 번만 초록 = 앱 만들 때마다 파랑딱지 = Claude가 함 빨강딱지 = 내가 클릭

😊 먼저 알아두세요

이 세 가지는 모두 무료로 시작할 수 있어요.

  • GitHub — 만든 코드를 보관하는 인터넷 창고
  • Vercel — 코드를 진짜 접속 주소로 만들어 주는 곳
  • Supabase — 입력한 내용을 인터넷에 저장하는 곳 (저장이 필요한 앱만)

아래 주황색(처음 한 번만)을 한 번 해두면, 그다음부터는 초록색(매번) 부분만 하면 됩니다.

🧰 처음 한 번만 — 준비하기 최초 1회

1
계정 3개 만들기 — 각 사이트에 들어가 무료로 가입합니다. 내가 가입
  • github.com · vercel.com · supabase.com
  • 💡 Vercel·Supabase는 “Continue with GitHub(깃허브로 계속)”로 가입하면 더 간단해요.
2
Claude에게 “필요한 도구를 설치해줘”라고 말합니다. Claude가 git과 GitHub 도구를 알아서 설치합니다. Claude가 함
3
GitHub 로그인 — Claude가 안내하면 화면에 뜬 코드를 입력하거나 Authorize를 눌러 로그인합니다. 내가 클릭
4
GitHub ↔ Vercel 연결 — Vercel에서 Continue with GitHub를 누르고 Authorize로 접근을 허용합니다. (한 번 연결하면 계속 유지돼요.) 내가 클릭
💡 가입·로그인·연결은 본인만 할 수 있어서 직접 클릭하셔야 해요. 어디를 누를지 헷갈리면 화면을 캡처해 Claude에게 보여주면 한 걸음씩 짚어줍니다.

⚡ 앱 만들 때마다 — 가장 간단한 요약 매번

Claude에게 이렇게 말하세요: “이거 인터넷에 올려줘.”
1
Claude가 코드를 GitHub에 자동으로 올립니다. Claude가 함
2
Claude가 Vercel 화면에서 어디를 누르라고 알려줍니다. 보통 Import → Deploy 두 번 클릭. 내가 클릭
3
끝! 진짜 인터넷 주소가 생깁니다. (예: ○○○.vercel.app) Claude가 확인
💡 데이터를 저장하는 앱이면 Supabase 단계가 하나 더 있어요. (아래 참고) 저장이 필요 없는 앱이면 이 요약만으로 끝납니다.

📦 GitHub — 코드 창고

이게 뭐예요? 만든 코드를 안전하게 보관하는 인터넷 창고예요. Vercel이 여기서 코드를 가져다 씁니다.

🟠 처음 한 번: 계정 가입 + 도구 설치 + 로그인 (위 “준비하기” 참고)
🟢 매번: 창고 만들기·코드 올리기는 전부 Claude가 자동으로 합니다. 내가 할 일: 없음
💡 아주 가끔 GitHub이 “본인 확인”을 요청할 수 있어요. 그러면 가입한 이메일로 온 숫자 코드를 화면에 입력하면 됩니다. Claude가 그때 짚어드려요.

🌐 Vercel — 진짜 주소 만들기 매번 조금

이게 뭐예요? GitHub의 코드를 진짜 접속 주소로 만들어 주는 곳이에요.

1
Claude가 알려주면 vercel.com/new로 들어갑니다. 내가 클릭
2
Import Git Repository 목록에서 방금 만든 앱 이름 옆의 Import 클릭. 내가 클릭
3
설정은 건드리지 말고 아래 검은 Deploy 버튼 클릭. 내가 클릭
4
Congratulations! 화면이 뜨면 성공. 가운데 미리보기 그림을 누르면 진짜 앱이 열립니다. Claude가 확인
💡 “2단계 보안(2FA) 설정할래?”가 나오면 지금은 Skip securing my account(건너뛰기)를 눌러도 됩니다. (나중에 켜는 걸 권장해요.) GitHub↔Vercel 연결을 한 번 해두면 다음부터는 위 4단계만 반복하면 됩니다.

🗄️ Supabase — 인터넷 저장 저장이 필요할 때만

이게 뭐예요? 입력한 내용을 인터넷에 저장해서, 핸드폰·태블릿·PC 어디서나 똑같이 보이게 해줘요. (메모·일정·기록처럼 저장이 필요한 앱에만 씁니다.)

1
Supabase에서 프로젝트를 하나 만듭니다. (처음 한 번만, 이후 재사용 가능) 내가 클릭
2
Claude가 저장할 표(테이블) 만드는 글(SQL)을 줍니다. Supabase의 SQL Editor에 붙여넣고 Run 클릭. 내가 클릭
3
초록색 Connect 버튼을 눌러 나오는 주소와 열쇠값(publishable key)을 복사해 Claude에게 붙여넣기. 내가 복사
4
Claude가 그 값을 앱에 넣고 연결·확인합니다. Claude가 함
⚠️ 열쇠값 중 publishable(또는 anon)만 씁니다 — 공개해도 안전한 값이에요. service_role / secret이라고 적힌 건 절대 복사하거나 남에게 주지 마세요. (헷갈리면 화면을 캡처해 Claude에게 물어보면 골라줍니다.)

🛟 막히거나 헷갈릴 때

  • 어떤 화면이든 캡처해서 Claude에게 보여주기 — 그러면 다음에 뭘 눌러야 할지 알려줍니다.
  • 캡처 방법(Windows): Windows + Shift + S → 마우스로 영역 드래그 → 대화창에 Ctrl + V
  • 창이 여러 개라 헷갈리면: Alt + Tab으로 창 바꾸기. 맨 위 탭 이름을 보고 고르면 됩니다.
  • “이거 맞아?” 싶으면 누르기 전에 먼저 캡처해서 물어보세요. 실수해도 대부분 되돌릴 수 있어요.

✏️ 이미 올린 앱을 고치고 싶을 때

Claude에게 이렇게 말하세요: “글자 크게 해줘 / 색 바꿔줘 / 항목 늘려줘” 등

Claude가 코드를 고쳐 GitHub에 다시 올리면, Vercel이 자동으로 인터넷 주소에 반영합니다. 이때는 대부분 아무것도 안 눌러도 됩니다. 😊

개발을 몰라도 괜찮아요 · 막히면 화면을 캡처해서 Claude에게 물어보세요
